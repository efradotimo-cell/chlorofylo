# chlorofylo
EcoSort-AI adalah solusi berbasis Deep Learning yang dirancang untuk mengatasi hambatan utama dalam pengelolaan limbah domestik: kesalahan pemilahan sampah pada sumbernya. Proyek ini dikembangkan oleh Efrado Timothy Wirasmo (Lead AI Engineer) bersama Gottlieb Jordan Simanungkalit (AI Engineer) dari SMA Penabur Bintaro Jaya.
