# Chlorofylo: MobileNetV2 FineTuned Waste Classifier 🌍
**Submission for SIEGA CUP 2026 - AI Engineering**

## 🚀 Project Overview
EcoSort-AI adalah sistem klasifikasi sampah otomatis menggunakan arsitektur **MobileNetV2**. Proyek ini dirancang untuk mendukung pengelolaan limbah yang lebih cerdas dan efisien.

### 🛠️ Tech Stack
* **Language:** Python 3.12
* **Framework:** TensorFlow / Keras
* **Architecture:** MobileNetV2 (Transfer Learning)
* **Optimization:** Adam Optimizer, Dropout 0.5
* **Deployment Ready:** TensorFlow Lite (.tflite) for Android

### 📈 Results
* **Accuracy:** 100% on training/validation (Demo Dataset)
* **Classes:** Plastic, Glass, Metal, Paper, Cardboard, Trash.

### 📂 Folder Structure
* `AI_SIEGA_CUP_2026.ipynb`: Core training logic.
* `model_waste_classifier.tflite`: Ready-to-use model for Android deployment.
* `predict.py`: Local inference script.
